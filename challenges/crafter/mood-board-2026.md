# Mood Board 2026

## Métadonnées

| Propriété | Valeur |
|-----------|--------|
| Type | Crafter |
| Durée solo | 1h |
| Durée Dojo | 1h |
| Difficulté | ⭐⭐ |
| Prérequis | Accès à un générateur d'images (Midjourney, DALL-E) |

---

## Description

> *Construisez un mood board prospectif assisté par IA — de la direction créative à la composition finale.*

"On veut un mood board pour le repositionnement de la marque." Ce challenge vous fait créer un workflow complet : brief créatif, génération de visuels, composition et note d'intention.

---

## Objectif du Challenge

**Ce que le participant doit produire :**

- [ ] Mood board visuel cohérent
- [ ] Note d'intention créative (150 mots)
- [ ] Workflow réutilisable et documenté

**Critères de réussite :**

- Les visuels sont cohérents entre eux
- La direction créative est claire
- La note d'intention est articulée

---

## Pistes de Solution

### Étape 1 : Brief créatif

**Prompt de cadrage :**
```
Je crée un mood board pour [projet/marque].

Contexte :
- Secteur : [industrie]
- Cible : [audience]
- Valeurs : [3-5 valeurs clés]
- Positionnement : [premium/accessible/innovant...]

Génère :
1. 5 mots-clés visuels
2. 3 palettes de couleurs (avec codes hex)
3. 3 styles typographiques recommandés
4. 5 références visuelles (artistes, marques)
5. 3 textures/matières évocatrices
```

### Étape 2 : Générer les visuels

**Prompts par catégorie :**

**Ambiance générale :**
```
[Mood keywords] aesthetic, editorial photography, 
soft lighting, [color palette], minimalist composition 
--ar 3:2 --style raw
```

**Textures :**
```
Close-up texture of [material], macro photography, 
[color] tones, high detail --ar 1:1
```

**Lifestyle :**
```
[Target persona] in [context], candid moment, 
[aesthetic style], natural lighting --ar 16:9
```

### Étape 3 : Composer

**Dans Figma/Canva :**
- Frame 1920x1080 ou A3
- Grille asymétrique (plus créatif)
- Mélanger les tailles d'images
- Ajouter palette + mots-clés

**Composition type :**
```
┌─────────────┬───────┐
│   Grande    │  Med  │
│   image     ├───────┤
│   (hero)    │ Small │
├─────┬───────┼───────┤
│ Med │ Small │  Med  │
├─────┴───────┴───────┤
│   Palette + Typo    │
└─────────────────────┘
```

### Étape 4 : Note d'intention

**Prompt :**
```
À partir de ce mood board [décrire les visuels], 
rédige une note d'intention créative de 150 mots :
- Direction artistique choisie
- Émotion visée
- Cohérence avec les valeurs
- Déclinaison sur différents supports
```

### Exemple : "Future of Wellness 2026"

**Mots-clés visuels :**
- Biophilic design
- Soft tech
- Mindful minimalism
- Organic geometry
- Warm neutrals

**Palette :**
- `#E8DFD5` Sable doux
- `#4A6741` Vert sauge
- `#B8860B` Or doux
- `#F5F5F0` Blanc cassé

### Tips & astuces

- 6-9 visuels max pour un mood board lisible
- Varier les formats (paysage, portrait, carré)
- La cohérence vient des couleurs et de l'ambiance

## Animation Dojo (1h, ~10 personnes)

### Objectifs pédagogiques

À la fin de l'atelier, les participants sauront :

- Définir une direction créative structurée
- Générer des visuels cohérents avec l'IA
- Composer un mood board équilibré
- Rédiger une note d'intention

### Prérequis participants

- Accès à un générateur d'images (Midjourney, DALL-E, Canva AI)
- Figma ou Canva pour la composition
- Un projet/marque en tête

### Matériel nécessaire

- [ ] Prompts par catégorie
- [ ] Templates Figma
- [ ] Exemples de mood boards professionnels
- [ ] Liste de mots-clés par secteur

### Déroulé minute par minute

| Temps | Phase | Animateur | Participants | Objectif |
|-------|-------|-----------|--------------|----------|
| 0-10 | Inspiration | Montre des mood boards professionnels | S'inspirent | Benchmark |
| 10-20 | Brief | Guide le prompt de cadrage | Définissent leur brief | Direction |
| 20-40 | Génération | Circule, aide sur les prompts | Créent les visuels | Production |
| 40-55 | Composition | Montre les principes de layout | Assemblent dans Figma | Design |
| 55-60 | Galerie | Anime le vote | Présentent et votent | Valorisation |

### Points d'attention animateur

**Pièges courants :**

- Trop de visuels → Maximum 9
- Incohérence de style → Utiliser les mêmes keywords
- Pas de direction claire → Revenir au brief

**Questions fréquentes :**

- "Mes images sont moches" → Itérer sur les prompts
- "Comment trouver le bon style ?" → S'inspirer de références
- "Je n'ai pas Midjourney" → DALL-E, Canva AI, ou curation Unsplash

### Variantes

**Sans génération IA (30 min) :**
- Curation sur Unsplash/Pinterest
- Focus sur la composition

**Version avancée (2h) :**
- Mood board animé (vidéo)
- Déclinaison multi-supports
- Présentation client
