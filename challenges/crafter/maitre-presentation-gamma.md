## Vision & Impact

Maîtriser le workflow complet de Gamma AI pour générer des présentations stratégiques percutantes. Cela permet de libérer du temps créatif pour la **narration** plutôt que pour la mise en forme.

## Le saviez-vous ?

L'IA permet de balayer 30 ans d'expertise traditionnelle en quelques minutes pour se concentrer sur la décision. Une présentation Gamma bien promptée vaut 10h de PowerPoint.

## Description

> *Créez des présentations professionnelles en quelques minutes avec Gamma AI.*

"Peux-tu faire 30 slides pour demain ?" Panique. Ce challenge vous fait maîtriser Gamma pour générer des présentations structurées, visuellement pro, en une fraction du temps habituel.

---

## Objectif du Challenge

**Ce que le participant doit produire :**

- [ ] 3 présentations de styles différents (pitch, rapport, formation)
- [ ] Workflow documenté de brief → présentation
- [ ] Bibliothèque de prompts par type

**Critères de réussite :**

- Présentations utilisables en l'état
- Temps de création < 15 min par deck
- Cohérence visuelle maintenue

## Pistes de Solution

### Les 3 types de présentations

**1. Pitch Deck (investisseurs/clients)**
```
Structure : Problème → Solution → Marché → Traction → Équipe → Ask

Prompt Gamma :
"Crée un pitch deck de 12 slides pour [produit].
Problème : [X]
Solution : [Y]
Style : Minimaliste, data-driven, Sequoia-style"
```

**2. Rapport/Bilan**
```
Structure : Contexte → Réalisations → Chiffres → Learnings → Next Steps

Prompt Gamma :
"Crée une présentation de bilan trimestriel.
Highlights : [liste]
Chiffres clés : [liste]
Style : Corporate moderne, graphiques clairs"
```

**3. Formation/Workshop**
```
Structure : Objectifs → Concepts → Exercices → Récap

Prompt Gamma :
"Crée une formation sur [sujet].
Durée : [X]h
Audience : [niveau]
Style : Pédagogique, visuel, mémorable"
```

### Workflow optimal

```
1. BRIEF : Écrire le message clé en 1 phrase
2. STRUCTURE : Lister les 5-7 points principaux
3. PROMPT : Envoyer à Gamma avec style souhaité
4. ITÉRATION : Ajuster slide par slide
5. POLISH : Ajouter visuels/data spécifiques
6. EXPORT : PDF ou lien de présentation
```

### Tips Gamma

```
✅ Ce qui marche bien :
- Brief structuré avec bullet points
- Références de style ("comme Apple", "style McKinsey")
- Données chiffrées à intégrer

❌ Ce qui marche moins bien :
- Brief trop vague ("fais une présentation sur l'IA")
- Trop de contenu par slide
- Pas de message clé défini
```
