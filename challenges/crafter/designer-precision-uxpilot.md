## Vision & Impact

Utiliser **UX Pilot** pour passer d'un design générique "Giga" à une expérience artisanale "Lovable". L'IA permet de générer des interfaces haute fidélité qui respectent les nuances esthétiques.

## Le saviez-vous ?

Le secret d'une interface qui ne ressemble pas à de l'IA réside dans le contrôle de la vibe et de l'intention créative. La direction artistique reste humaine.

## Description

> *Créez des interfaces uniques qui ne ressemblent pas à du "généré par IA".*

"Ça fait template." La critique ultime. Ce challenge vous fait maîtriser les techniques pour que vos designs générés par IA aient une personnalité unique et reconnaissable.

---

## Objectif du Challenge

**Ce que le participant doit produire :**

- [ ] 3 interfaces avec une identité visuelle distincte
- [ ] Guidelines pour éviter le "look IA"
- [ ] Avant/Après montrant la différence

**Critères de réussite :**

- Interfaces reconnaissables et uniques
- Cohérence avec une marque fictive ou réelle
- Personne ne devine que c'est généré par IA

## Pistes de Solution

### Les signes d'un design "IA générique"

```
❌ À ÉVITER :
- Gradients bleu-violet partout
- Illustrations style "Blush/Humaaans"
- Rounded corners excessifs
- Ombres drop shadow par défaut
- Palette trop safe (bleu corporate)
- Espacement uniforme partout
- Typo système (Inter, sans personnalité)
```

### Comment personnaliser

```
✅ PERSONNALISATION :

1. PALETTE UNIQUE
   - Choisir une couleur primaire distinctive
   - Créer des variations subtiles
   - Utiliser des accents inattendus

2. TYPOGRAPHIE CARACTÈRE
   - Choisir une font avec personnalité
   - Créer une hiérarchie distinctive
   - Oser les contrastes de poids

3. ESPACEMENT INTENTIONNEL
   - Varier le rythme
   - Créer des zones de respiration
   - Casser la grille parfois

4. DÉTAILS CRAFT
   - Micro-interactions uniques
   - Illustrations custom ou photos
   - Textures subtiles
```

### Prompt pour design unique

```
Génère une interface [type] qui NE ressemble PAS à du design IA générique.

Identité de marque :
- Personnalité : [3 adjectifs]
- Couleur signature : [hex code]
- Font caractère : [nom]
- Inspiration : [marque ou style]

ÉVITE ABSOLUMENT :
- Gradients bleu-violet
- Illustrations génériques
- Espacement uniforme
- Ombres par défaut

RECHERCHE :
- Une signature visuelle reconnaissable
- Des détails qui surprennent
- De la personnalité dans chaque élément
```

### Les 3 identités à créer

**1. Marque "Craft" (artisanale)**
```
- Couleurs : Terracotta, crème, vert sauge
- Typo : Serif élégante + Sans humaniste
- Vibe : Calme, premium, naturel
```

**2. Marque "Tech" (moderne)**
```
- Couleurs : Noir, blanc, accent néon
- Typo : Geometric bold + Mono
- Vibe : Sharp, minimal, futuriste
```

**3. Marque "Fun" (playful)**
```
- Couleurs : Palette pop, jaune vif
- Typo : Rounded bold + Hand-drawn
- Vibe : Joyeux, accessible, décalé
```

## Animation Dojo (1h30)

| Temps | Phase | Activité |
|-------|-------|----------|
| 0-15 | Intro | "Ça fait IA" - pourquoi et comment éviter |
| 15-30 | Analyse | Décortiquer des designs uniques vs génériques |
| 30-60 | Création | Chacun crée une interface distinctive |
| 60-80 | Critique | Peer review : "Est-ce que ça fait IA ?" |
| 80-90 | Clôture | Guidelines personnelles |
