# L'Expert Identité Visuelle

## Métadonnées

| Propriété | Valeur |
|-----------|--------|
| Type | Crafter |
| Durée solo | 1h |
| Durée Dojo | 1h |
| Difficulté | ⭐⭐ |
| Prérequis | Aucun |

---

## Description

> *Créez un assistant expert en branding qui vous guide dans la définition de votre identité visuelle : mots-clés, palette couleurs, typographies.*

"On veut un logo moderne et chaleureux." Mais ça veut dire quoi concrètement ? Ce challenge vous fait créer un directeur artistique IA qui traduit vos valeurs en choix visuels cohérents.

---

## Objectif du Challenge

**Ce que le participant doit produire :**

- [ ] Brief identité visuelle complet
- [ ] Palette couleurs avec codes hex
- [ ] Recommandations typographiques
- [ ] (Bonus) Application sur une maquette

**Critères de réussite :**

- Les mots-clés visuels sont définis et justifiés
- La palette est cohérente avec les valeurs de la marque
- Les recommandations sont applicables immédiatement

---

## Pistes de Solution

### Processus de l'assistant

1. **Découverte** — Mission, cible, valeurs, concurrents
2. **Mots-clés visuels** — 5-7 mots qui guident les choix
3. **Palette couleurs** — Primaire, secondaire, accents, neutres
4. **Typographie** — Titrage, corps, accent
5. **Application** — Déclinaison sur les supports

### Prompt système suggéré

```
Tu es un directeur artistique senior spécialisé en branding avec 15 ans d'expérience.

## Ta mission
Accompagner la définition d'une identité visuelle cohérente.

## Ton processus

### Phase 1 : Découverte
- Mission de la marque ?
- Clients cibles ?
- Valeurs fondamentales ?
- Concurrents ?
- Identité existante ?

### Phase 2 : Mots-clés visuels
5-7 mots qui guideront toutes les décisions.
Ex : "Chaleureux, Moderne, Accessible"

### Phase 3 : Palette
- Primaire (identité forte)
- Secondaire (complémentaire)
- Accents (CTA, alertes)
- Neutres (texte, fond)

### Phase 4 : Typographie
- Titrage (personnalité)
- Corps (lisibilité)
- Accent (highlights)

Commence par : "Parle-moi de la marque : mission, cible, valeurs..."
```

### Format du brief

```markdown
# Brief Identité Visuelle - [Marque]

## 1. Essence de marque
**Mission** : [En une phrase]
**Valeurs** : [3-5 valeurs]
**Personnalité** : [Traits de caractère]
**Positionnement** : [Vs concurrents]

## 2. Mots-clés visuels

| Mot-clé | Traduction visuelle |
|---------|---------------------|
| Chaleureux | Couleurs chaudes, formes arrondies |
| Moderne | Typo sans-serif, espaces généreux |

## 3. Palette de couleurs

### Primaire
- **Nom** : Bleu Confiance
- **Hex** : #2E5BFF
- **Usage** : Logo, CTA principaux

### Secondaire
- **Nom** : Corail Énergie
- **Hex** : #FF6B6B
- **Usage** : Accents, hover

### Neutres
- Texte : #1A1A2E
- Fond : #F5F5F7

## 4. Typographie

### Titrage
- **Police** : Inter Bold
- **Usage** : H1, H2, CTA

### Corps
- **Police** : Inter Regular
- **Taille** : 16px / 1.5

## 5. Éléments graphiques
- Formes : Coins arrondis (8px)
- Icônes : Style ligne, 2px
- Photos : Naturelles, lumière chaude
```

### Tips & astuces

- Demander des exemples de marques admirées
- Toujours justifier les choix par les valeurs
- Proposer des alternatives si hésitation

## Animation Dojo (1h, ~10 personnes)

### Objectifs pédagogiques

À la fin de l'atelier, les participants sauront :

- Définir des mots-clés visuels à partir de valeurs
- Construire une palette cohérente
- Choisir des typographies adaptées
- Produire un mini-brief identité

### Prérequis participants

- Avoir une marque/projet en tête
- Accès à un chat IA

### Matériel nécessaire

- [ ] Prompt système
- [ ] Exemples d'identités visuelles fortes
- [ ] Template de brief
- [ ] Outil de palette (Coolors, Adobe Color)

### Déroulé minute par minute

| Temps | Phase | Animateur | Participants | Objectif |
|-------|-------|-----------|--------------|----------|
| 0-10 | Intro | Exemples d'identités fortes (Apple, Spotify) | Comprennent l'impact | Inspiration |
| 10-20 | Découverte | Guide les questions | Définissent leur marque | Clarification |
| 20-35 | Palette & typo | Aide sur les associations | Génèrent les recommandations | Production |
| 35-50 | Application | Montre comment décliner | Appliquent sur un template | Concrétisation |
| 50-60 | Partage | Fait présenter 3 briefs | Présentent leurs choix | Valorisation |

### Points d'attention animateur

**Pièges courants :**

- Trop de couleurs → Maximum 5-6 couleurs
- Mots-clés vagues → Demander la traduction visuelle
- Incohérence valeurs/visuels → "Pourquoi ce choix ?"

**Questions fréquentes :**

- "Comment choisir entre 2 palettes ?" → Tester sur un support réel
- "Et si ma marque évolue ?" → Construire un système flexible
- "Faut-il un designer ensuite ?" → Le brief guide le designer

### Variantes

**Version express (30 min) :**
- Focus sur palette + 3 mots-clés
- Pas d'application

**Version avancée (2h) :**
- Maquette complète (landing page)
- Déclinaison multi-supports
