## Description

> *Créez des campagnes visuelles immersives avec une cohérence de monde.*

Une image c'est bien. Un univers c'est mieux. Ce challenge vous fait créer une campagne complète avec personnages récurrents, environnements cohérents et storytelling visuel.

---

## Objectif du Challenge

**Ce que le participant doit produire :**

- [ ] Bible visuelle d'un univers (personnages, lieux, ambiance)
- [ ] 10 images cohérentes formant une histoire
- [ ] Guidelines de reproduction

**Critères de réussite :**

- Personnages reconnaissables d'une image à l'autre
- Univers cohérent et immersif
- Utilisable pour une vraie campagne

---

## Pistes de Solution

### Structure d'un univers visuel

```
UNIVERS = Personnages + Lieux + Ambiance + Règles

PERSONNAGES
- Design détaillé (vêtements, traits, style)
- Poses et expressions clés
- Variations (contextes différents)

LIEUX
- Environnements récurrents
- Éclairage signature
- Détails distinctifs

AMBIANCE
- Palette de couleurs
- Style artistique (réaliste, stylisé...)
- Mood général

RÈGLES
- Ce qui est OK / pas OK
- Limites de variation
- Cohérence à maintenir
```

### Prompt de création de personnage

```
Crée un personnage pour une campagne marketing :

Concept : [description du personnage]

Détails à définir :
- Âge apparent : X ans
- Style vestimentaire : [description]
- Traits distinctifs : [éléments reconnaissables]
- Personnalité visuelle : [3 adjectifs]

Style : [photoréaliste / illustration / 3D / etc.]
Éclairage : [type]
Palette dominante : [couleurs]

Génère 4 poses différentes pour établir le personnage :
1. Portrait face
2. Portrait 3/4
3. En action
4. En contexte (environnement)
```

### Maintenir la cohérence

```
TECHNIQUES DE COHÉRENCE :

1. SEED FIXE (Midjourney)
   --seed [number] pour reproduire un style

2. STYLE REFERENCE
   --sref [image_url] pour maintenir l'esthétique

3. CHARACTER REFERENCE
   --cref [image_url] pour garder le personnage

4. PROMPT TEMPLATE
   Utiliser exactement les mêmes descripteurs de base
```

### Workflow campagne

```
ÉTAPE 1 : BIBLE VISUELLE
├── Créer 1 personnage principal
├── Créer 2-3 personnages secondaires
├── Définir 3 lieux clés
└── Fixer la palette et le style

ÉTAPE 2 : KEYFRAMES
├── 3 images clés qui racontent l'histoire
├── Validation de la cohérence
└── Ajustements

ÉTAPE 3 : DÉCLINAISONS
├── Variations pour chaque canal
├── Formats différents
└── Adaptations contextuelles

ÉTAPE 4 : GUIDELINES
├── Documenter les prompts
├── Sauvegarder les seeds/refs
└── Créer le guide de reproduction
```

### Les 10 images à créer

```
HISTOIRE EN 10 IMAGES :

1. Introduction du héros
2. Son environnement quotidien
3. Le problème/défi
4. La découverte (du produit/solution)
5. Premier essai
6. Obstacle
7. Persévérance
8. Réussite
9. Transformation
10. Nouvelle vie / CTA
```

## Animation Dojo (2h)

| Temps | Phase | Activité |
|-------|-------|----------|
| 0-15 | Intro | De l'image à l'univers |
| 15-35 | Personnage | Créer un personnage cohérent |
| 35-55 | Environnement | Créer les lieux |
| 55-90 | Histoire | Générer la séquence de 10 images |
| 90-110 | Revue | Cohérence check |
| 110-120 | Clôture | Créer les guidelines |

### Points d'attention

- La cohérence demande de l'itération
- Documenter TOUT (prompts, seeds, refs)
- Commencer simple, complexifier ensuite
